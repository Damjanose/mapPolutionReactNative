# Map Polution React Native

Clone the project ``` git clone ```
Change directory ``` cd ./mapPolutionReactNative ```
Install node modules ``` npm i ```
Run project ``` npm start ```
## 🚀 How to use

```sh
npx create-react-native-app -t with-router
```
