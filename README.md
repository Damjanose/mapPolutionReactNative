# Map Polution React Native

Clone the project ```sh git clone ```
Change directory ```sh cd ./mapPolutionReactNative ```
Install node modules ```sh npm i ```
Run project ```sh npm start ```

## 🚀 How to use

```sh
npx create-react-native-app -t with-router
```
